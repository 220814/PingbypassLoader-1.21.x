# PingbypassLoader
- Actually, it's a loader used to run Phobot plugins.
- It does not contain any actual pingbypass codes.
- It's unstable because it has been modified and the packets have changed too much over the past 5 months.
- Version 1.21.1 - fabric
- checking the code to make sure it doesn't contain any malicious code.
- You yourself will test and build it.
- Once the build is complete, there will be 3 jar files, and you need to put them all into the mods folder.
- Finally, enjoy :)
