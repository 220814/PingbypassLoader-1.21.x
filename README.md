# PingbypassLoader
- It's called a pingbypass loader, but it's not; it's a phobot loader. It's used to run 3arthqu4ke's phobot plugins on version 1.21.1.

# Instructions for use:
- I will not upload any pre-built files here for security reasons.
- You will be the one to check the entire source code and build it yourself.
- After building, you will have 3 jars. All you need to do is put them all in the mods folder (it can be run with Mio).
- enjoy :)

# warning : 
- The code has been modified or changed too much due to network and packet changes from 1.20.4 to 1.21.1, so it will not work perfectly.
- It's called pingbypass but doesn't contain any actual pingbypass functionality.
- Run it with 1.21.1 - fabric.
- I think you shouldn't run this because I haven't even updated Phobot's plugins to version 1.21.1 yet.
- Open the issue if the code has errors or crashes.
